# Only Murders: Arconia Investigation

A replayable co-op mystery game inspired by Only Murders in the Building.

## Run

Open `/Users/julvie/Documents/Codex Project/Board Game/index.html` in a modern browser.

## Current UX

- **Game setup** is now in a **top-right menu button** (`Game Setup`).
- Setup includes player count, difficulty, character card selection, and new game.
- No end-turn button: turns advance automatically right after movement.

## Current Rules

- No stolen-clue system.
- No confirmed-insights panel.
- Roll die, move on the 4-sided board, gather clues, and build evidence scores.
- Make a final accusation with all 4 parts:
  - Suspect
  - Murder Object
  - Apartment
  - Motive

## Character Cards

- Mabel
- Charles
- Oliver
- Detective Williams

Each card has custom illustrated SVG portrait, stats, and a passive ability.
